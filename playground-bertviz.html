Attention — Qui regarde qui ?

Chaque ligne relie un token (à gauche) au token qu'il regarde (à droite). Plus la ligne est épaisse, plus l'attention est forte. Clique sur un token pour voir ses poids en détail.

Tape un nom
Change le nom pour voir comment l'attention se redistribue. Un token ne peut jamais regarder le futur — c'est le masque causal.
Les noms des têtes (Ancrage, Contexte…) décrivent des comportements observés après un entraînement. Mais un autre entraînement pourrait assigner ces rôles à des têtes différentes — c'est le hasard de l'initialisation. Ce qui est stable, c'est que ces types de comportements émergent toujours.
